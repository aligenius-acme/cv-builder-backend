## Technical Implementation

### Component Usage
All templates use shared components:
- `Header` - Name and contact display
- `SectionHeader` - Consistent section titles (5 variants)
- `SkillsSection` - Skills display (grid, list, pills, inline, columns)
- `ExperienceSection` - Work experience (3 layouts)

- `ProjectsSection` - Project showcase with tech stack
- `CertificationsSection` - Certifications list (4 layouts)
