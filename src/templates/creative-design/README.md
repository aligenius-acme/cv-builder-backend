# Creative & Design Templates

**15 Bold, Visually Striking Templates for Creative Professionals**

---

## Overview

The Creative & Design category features templates designed for professionals in creative fields who want to showcase their personality, portfolio, and visual design skills. These templates emphasize bold design, strong use of color, and creative layouts while maintaining reasonable ATS compatibility (60-75%).

### Key Features

- **Bold, Creative Layouts**: Two-column sidebars and banner headers
- **Strong Visual Hierarchy**: Clear section organization with creative flair
- **Portfolio Emphasis**: Prominent project/portfolio sections
- **Color Customization**: Full color palette support
- **Visual-First Design**: Designed to stand out visually (ATS: 60-75%)

## Design Characteristics

### Color Usage
- **Bold Colors**: Primary color used prominently in headers, sidebars, and sections
- **Backgrounds**: Colored backgrounds for sidebar, banner headers, and section boxes
- **Contrast**: High contrast between colored areas and white text
- **Gradients**: Simulated through layered colors and opacity

### Typography
- **Fonts**: Helvetica, Arial, sans-serif
- **Name Size**: 20-30pt, bold to extra bold (700-900)
- **Title Size**: 11-14pt, uppercase with letter spacing
- **Body**: 10-11pt, readable line height (1.6-1.7)
- **Section Headers**: 12-15pt, bold, uppercase

## Usage Guidelines

### When to Use Creative Templates

**Best For**:
- Creative industries (design, media, marketing)
- Portfolio-heavy roles
- Visual design positions
- Brand and marketing roles
- Content creation and social media

**Consider ATS Templates Instead If**:
- Applying to very traditional industries (finance, legal, government)
- Company specifically requests "ATS-friendly" resume
- Minimal portfolio to showcase
- Conservative corporate environment

### Customization Tips

1. **Colors**: All templates support custom color palettes
2. **Portfolio Links**: Make website/portfolio URL prominent
3. **Projects**: Use the projects section extensively
4. **Skills**: Visual skill displays (pills, grids) are prominent
5. **Photos**: Not included, but can add to sidebar templates

### Export Recommendations

- **PDF Export**: Recommended for all templates (preserves visual design)
- **DOCX Export**: Use for banner header templates when ATS compatibility is important
- **Online Applications**: PDF maintains formatting across all systems

## Implementation Notes

### PDF Generation
- Uses React components with inline styles
- Full color customization support
- Flexbox layouts for two-column designs
- Responsive typography

### DOCX Generation
- Uses `docx` library with ATS-safe structure
- Single-column layouts only
- Standard fonts (Arial, Calibri)
- Proper heading hierarchy
- No tables for layout

### Shared Components Used
- `Header.tsx` - Banner and split variants
- `SectionHeader.tsx` - Boxed, filled, underline variants
- `SkillsSection.tsx` - Pills, grid, list layouts
- `ProjectsSection.tsx` - Detailed project displays
- `ExperienceSection.tsx` - Standard and boxed layouts
- `EducationSection.tsx` - Default and compact layouts
- `ContactInfo.tsx` - Vertical and horizontal layouts
